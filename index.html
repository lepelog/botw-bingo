Bingo Save File

  • You start on top of the Temple of Time where you get the Glider.
  • You are not allowed to have any items in the Save File initially. So drop all your items.
  • Your Save File must not have left the Plateau yet.
  • Don't do any unnecessary steps to make your save file.
  • Your Save File must have done 4 shrines and must not have used those Soul Orbs yet.

Rules

  • Amiibos are not allowed.
  • DLC is only allowed if all racers agree to it
  • "Complete x shrines" includes the 4 Plateau shrines.
  • "Obtain x armor" is the Chest-piece unless "set" is specified.
  • The Horse God is not a Great Fairy.
  • Items obtained for goals must be kept until the end.
